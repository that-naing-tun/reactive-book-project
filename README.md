# reactive-book-project
# 📚 Book Reading List App

A mobile application built with **React Native (Expo)** that allows users to manage their personal reading list. Users can create, view, and delete books. The app uses **Appwrite Authentication, TablesDB, and Realtime Database** to store and synchronize data.

---

## 🚀 Features

- User authentication (Register/Login/Logout)
- Create new books
- View personal book list
- View book details
- Delete books
- Realtime updates using Appwrite Realtime
- User-based data access
- Protected routes
- Responsive UI with custom themed components

---

## 🛠️ Technologies Used

### Frontend

- React Native
- Expo
- Expo Router
- JavaScript
- React Hooks
- Context API

### Backend / Database

- Appwrite
  - Authentication
  - TablesDB
  - Realtime API
  - Permissions
